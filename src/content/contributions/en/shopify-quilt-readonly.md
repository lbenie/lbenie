Contributed to Shopify's Quilt library, a collection of JavaScript/TypeScript packages that power Shopify's internal tools and applications. This pull request added support for readonly typed schemas, enhancing type safety and enabling better immutability patterns in TypeScript projects.

The feature aimed to help developers working with Quilt to enforce immutability at the type level, preventing accidental mutations and improving code reliability. This is particularly valuable in large-scale applications where data consistency and predictability are critical.
