Contributed a feature enhancement to Shopify's Polaris React IndexTable component by adding the ability to clear all selections at once. This functionality improves the user experience when working with bulk actions and large datasets.

The feature allows merchants to quickly deselect all items in a table with a single action, streamlining workflows that involve selective bulk operations. This is particularly useful in scenarios where users need to reset their selection state without manually deselecting individual items.
